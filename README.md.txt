📊 Exploratory Data Analysis (EDA) – Power BI Project

📌 Project Overview
This project presents an Exploratory Data Analysis (EDA) dashboard built using Power BI.  
The objective was to transform raw data into meaningful insights through data cleaning, modeling, and visualization.

The dashboard provides interactive analysis of key performance indicators to support data-driven decision-making.


🎯 Business Objectives
- Analyze overall performance trends
- Identify top-performing categories/products
- Evaluate key metrics and growth patterns
- Build an interactive dashboard for insight exploration


🛠 Tools & Technologies Used
- Power BI
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)
- Microsoft Excel (Data Source)

🔎 Data Preparation Process
- Cleaned and transformed raw dataset using Power Query
- Removed duplicates and handled missing values
- Created calculated measures using DAX
- Built relationships between data tables


📊 Dashboard Features
- Interactive slicers for dynamic filtering
- KPI indicators for performance tracking
- Trend analysis visuals
- Category and segment performance breakdown


📈 Key Insights
- Identified performance trends across time periods
- Highlighted top revenue-generating categories
- Provided structured data visualization for strategic decisions

💡 Business Recommendations
- Focus on high-performing segments to maximize revenue
- Monitor declining trends to implement corrective actions
- Use dashboard insights for quarterly planning
